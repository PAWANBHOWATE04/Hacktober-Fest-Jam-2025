# Code of Conduct

We are committed to a friendly, safe, and welcoming environment for all.

- Be respectful and constructive.
- Assume good intent; ask questions.
- No harassment, discrimination, or exclusionary behavior.

If you experience or witness unacceptable behavior, contact the organizers privately.  
Participation implies agreement with this Code of Conduct.
